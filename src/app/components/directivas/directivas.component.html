directivas estructurales

  • *ngIF

    • Permite incluir una plantilla de forma condicional en base al valor de una expresión forzada a booleana. Cuando la expresión se evalúa como verdadera, Angular representa/renderiza la plantilla proporcionada en una cláusula "then", y cuando es falsa o nula, Angular representa la plantilla proporcionada en una cláusula opcional "else".   • *ngFor

    • Permite representa una plantilla para cada elemento dentro de una colección. La directiva se coloca en un elemento, que se convierte en el padre de las plantillas clonadas. Ejemplo:


    Nombre Apellidos Edad Correo
    {{ iter_person.nombre }} {{ iter_person.apellido }} {{ iter_person.edad }} {{ iter_person.correo }}

  • ngSwitch

    • La directiva [ngSwitch] en un contenedor especifica una expresión con la que comparar. Las expresiones para hacer coincidir las proporcionan las directivas ngSwitchCase en las vistas dentro del contenedor.

    • Se representa cada vista que coincide.
    • Si no hay coincidencias, se representa una vista con la directiva ngSwitchDefault.
    • Los elementos dentro de la instrucción [NgSwitch] pero fuera de cualquier directiva NgSwitchCase o ngSwitchDefault se conservan en la ubicación.
    Ejemplo:

    Usuario Registrado: Hola {{ persona.nombre }}!

    Usuario Registrado: Hola {{ persona.nombre }}!

    Usuario Registrado: Hola {{ persona.nombre }}!

    Usuario Registrado: Hola {{ persona.nombre }}!

    Usuario Desconocido ... Mucho gusto {{ persona.nombre }}!

directivas de atributo

  • ngClass

    • Agrega y elimina clases CSS en un elemento HTML.


  • ngStyle

    • Actualiza los estilos del elemento HTML que lo contiene. Establece una o más propiedades de estilo, especificadas como pares clave-valor separados por dos puntos. La clave es un nombre de estilo, con un sufijo. El valor es una expresión a evaluar. Si el resultado de la evaluación es nulo, se elimina el estilo correspondiente.


  • ngPlural

    • Agrega/elimina subárboles DOM en función de un valor numérico. Diseñado para la pluralización.
